# 🚖 Uber Clone | React Native App

Aplicación móvil desarrollada con **React Native** que replica las funcionalidades principales de una plataforma de transporte tipo Uber, permitiendo solicitar viajes, visualizar ubicaciones y gestionar usuarios en tiempo real.

---

## 📌 Descripción

**Uber Clone** es una app móvil multiplataforma diseñada para simular un sistema de transporte bajo demanda. Ofrece una experiencia fluida e interactiva, integrando geolocalización, autenticación y comunicación con servicios backend.

La aplicación está enfocada en demostrar el desarrollo de apps modernas utilizando tecnologías actuales del ecosistema JavaScript.

---

## 🚀 Características

- 📱 Aplicación multiplataforma (Android / iOS)
- 🚖 Solicitud de viajes en tiempo real
- 📍 Geolocalización y mapas
- 🔐 Autenticación de usuarios
- ⚡ Interfaz rápida y responsiva
- 🔄 Comunicación con API
- 💾 Manejo de estado dinámico

---

## 🛠️ Tecnologías utilizadas

### Frontend
- **React Native**
- **Expo**
- **JavaScript (ES6+)**

### Backend / Servicios
- **API REST**
- **Firebase / Backend personalizado (opcional)**

### Integraciones
- **Google Maps API** → Mapas y ubicación
- **Stripe (opcional)** → Pagos

## ⚙️ Instalación

1. Clonar el repositorio:
```bash
git clone https://github.com/isairey/App-de-Uber-Clone.git
```
Acceder al proyecto:
```
cd App-de-Uber-Clone
```
Instalar dependencias:
```
npm install
```
---

## ▶️ Ejecución
```
npx expo start
```

- 💻 Ejecuta en emulador Android/iOS
